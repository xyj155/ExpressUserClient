####    快递小哥为我参加浙江省新苗项目接下的新苗项目，然后和一个老年痴呆的导师一起商量，我写前端，我写后台，我写接口，我写后台管理程序，然后导师看戏。没错，就是这样子，臭不要脸的那种！！！！！

####    APP里面采用的是MVP的架构模式，什么叫MVP呢，自己百度，或者，我教你百度？？嘿嘿！！
